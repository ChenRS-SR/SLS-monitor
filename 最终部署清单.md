## 🚀 快速部署流程

### 1. 复制整个SLS文件夹
```bash
# 压缩整个SLS目录
7z a SLS_v2.0_完整版.zip SLS/
# 或
Compress-Archive -Path SLS -DestinationPath SLS_v2.0_完整版.zip
```

### 2. 在目标设备上解压并安装
```bash
# 解压项目
unzip SLS_v2.0_完整版.zip

# 进入项目目录
cd SLS

# 方法1: 使用快速启动脚本 (推荐)
启动SLS.bat

# 方法2: 手动安装依赖
pip install -r requirements.txt
python 部署验证.py
python run.py
```
