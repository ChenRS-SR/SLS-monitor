# SLS监控系统状态机模式说明

## 🎯 两种状态机模式

现在系统支持两种状态机模式，你可以在界面上选择适合你设备的模式：

### 🔵 简单模式（单向刮刀）
**适用场景**：双粉缸SLS打印机，刮刀单向运动
- 第一层：从左到右运动
- 第二层：从右到左运动  
- 无需返回平整

```
📸 拍摄时机：
before → 刮刀运动 → after
  ↓                  ↑
idle → motion → idle
  ↑              ↓
  ← 下一层 ← ← ← ←
```

**状态转换**：
1. `idle` → 检测到振动 → `motion` + 拍摄before图像
2. `motion` → 振动停止稳定 → 拍摄after图像 → `idle`（进入下一层）

**时间参数**：
- `debounce_time`：振动停止确认时间（默认1.0秒）

### 🟠 复杂模式（双向刮刀）  
**适用场景**：传统SLS打印机，需要铺粉+平整两个动作
- 第一次运动：铺粉
- 短暂停顿：准备或调整
- 第二次运动：平整粉层

```
📸 拍摄时机：
before → 第一次运动 → 停顿 → 第二次运动 → after
  ↓                                          ↑
idle → first_motion → between_motions → second_motion → idle
  ↑                                                      ↓
  ← ← ← ← ← ← 下一层 ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

**状态转换**：
1. `idle` → 检测到振动 → `first_motion` + 拍摄before图像
2. `first_motion` → 振动停止 → `between_motions`
3. `between_motions` → 检测到振动 → `second_motion`
4. `between_motions` → 超时 → 直接拍摄after + 下一层
5. `second_motion` → 振动停止稳定 → 拍摄after图像 → `idle`（进入下一层）

**时间参数**：
- `debounce_time`：第一次运动停止确认时间（默认1.0秒）
- `between_motions_timeout`：等待第二次运动的最大时间（默认30秒）
- `second_motion_settle_time`：第二次运动停止确认时间（默认2.0秒）

## 🛠️ 如何选择模式

### 选择简单模式的情况：
- ✅ 你的设备有两个粉缸（左右各一个）
- ✅ 刮刀只做单向运动（左→右 或 右→左）
- ✅ 每层只有一次刮刀动作
- ✅ 不需要返回平整粉层

### 选择复杂模式的情况：
- ✅ 你的设备只有一个粉缸
- ✅ 刮刀需要往返运动
- ✅ 每层有两次动作：铺粉 + 平整
- ✅ 需要检测两个独立的运动阶段

## 📊 界面操作

1. **模式切换**：在控制面板的调试区域选择对应的单选按钮
   - 🔘 简单模式(单向刮刀)
   - 🔘 复杂模式(双向刮刀)

2. **实时状态**：状态机当前状态会在界面上显示
   - 简单模式：`idle` → `motion` → `idle`
   - 复杂模式：`idle` → `first_motion` → `between_motions` → `second_motion` → `idle`

3. **调试信息**：所有状态转换都会在控制台和日志文件中记录
   - `[简单模式]` 前缀：简单状态机的日志
   - `[复杂模式]` 前缀：复杂状态机的日志

## 🎯 推荐设置

### 对于你的设备（双粉缸）：
**推荐使用简单模式**
- 设置振动阈值：0.5-1.0（根据实际情况调整）
- 防抖时间：1.0-2.0秒
- 开启测试模式先验证状态转换

### 参数调整建议：
1. **振动阈值太低**：会误触发，产生过多false positive
2. **振动阈值太高**：检测不到真实运动
3. **防抖时间太短**：运动过程中的短暂停顿会被误认为结束
4. **防抖时间太长**：运动结束后等待时间过长

## 🔧 故障排除

### 状态机不转换：
1. 检查振动检测是否正常工作（控制台有触发信息）
2. 检查阈值设置是否合适
3. 查看日志文件中的详细状态转换信息
4. 尝试调整防抖时间参数

### 误触发问题：
1. 提高振动阈值
2. 检查设备是否有其他振动源干扰
3. 调整传感器位置或敏感度

### 检测延迟：
1. 降低振动阈值
2. 减少防抖时间
3. 检查传感器连接和数据更新频率
